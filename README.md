# Offline Expense Tracker

A Flutter offline-first expense tracker with Riverpod + Hive, analytics charts, budgets, recurring transactions, local exports (PDF/CSV), local backup/restore, and app lock (PIN/Biometric).

## Highlights
- 100% local persistent data (Hive)
- Feature-based clean structure
- Riverpod state management
- Charts with fl_chart
- PDF + CSV export
- Local notifications + local auth
- Dark/Light/System theming

## Run
1. Install Flutter stable SDK.
2. `flutter pub get`
3. `flutter run`
